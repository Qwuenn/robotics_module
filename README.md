## Building your site

```bash
# Build static files with Jigsaw
./vendor/bin/jigsaw build

# Preview static site using Jigsaw
./vendor/bin/jigsaw serve 

# Compile assets with Laravel Mix
# options: dev, staging, prod
npm run dev

# Preview site with Browsersync
npm run watch

# Run git log with --abbrev-commit --pretty=oneline
npm run git-log

# Deploy your site to github pages
# WARNING: Make sure you have pushed all commited changes to origin before running the command
npm run deploy
```
